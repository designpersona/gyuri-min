# Gyuri Min - Portfolio

Personal portfolio website for Gyuri Min.

The site presents selected works across brand, UX, and system-oriented design,
and is structured as a curated archive rather than a linear showcase.

## Stack

- HTML
- CSS
- JavaScript
- Tailwind CDN for utility classes

## Deployment

- The site is static and can be deployed with GitHub Pages from the repository
  root.
- `.nojekyll` is included so GitHub Pages serves static assets without Jekyll
  processing.
- CSS and JavaScript files are cache-busted with query versions in `index.html`.
- Large video assets are kept in `assets/projects`; check them before publishing
  if repository size or page loading speed becomes a concern.

## Privacy

This repository is intended for public portfolio deployment. Keep private
credentials, unpublished client materials, local environment files, and
machine-specific paths out of the repository.
